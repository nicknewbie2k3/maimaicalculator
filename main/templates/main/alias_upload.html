File Format Information

To bulk upload aliases, please use a JSON file with the following format:

JSON Format Options:
Single Song Format:
{
  "chart_name": "Base Song Title (without [DX]/[STD])",
  "chart_alias": ["alias1", "alias2", "alias3"]
}
Multiple Songs Format:
[
  {
    "chart_name": "First Song Title",
    "chart_alias": ["alias1", "alias2"]
  },
  {
    "chart_name": "Second Song Title", 
    "chart_alias": ["alias3", "alias4"]
  }
]
Examples:
Single Song: 
{
  "chart_name": "熱異常",
  "chart_alias": ["heat abnormal", "thermal anomaly"]
}
Multiple Songs: 
[
  {
    "chart_name": "熱異常",
    "chart_alias": ["heat abnormal"]
  },
  {
    "chart_name": "熱異常指數", 
    "chart_alias": ["heat abnormal index"]
  }
]
Important Notes:
  • chart_name can be the base song title (system will auto-match [DX]/[STD] variants)
  • chart_alias should be an array of strings (even if just one alias)
  • You can upload single song or multiple songs in one file
  • If both [DX] and [STD] versions exist, aliases will be added to both automatically
  • Aliases will be added to existing aliases (not replace them)
  • Duplicate aliases will be ignored automatically
  • Invalid entries will be skipped with detailed error messages
Chart Name Matching:

Smart Matching: You can use base song names without chart type tags!

  • "熱異常" → automatically finds "熱異常 [DX]" and/or "熱異常 [STD]"
  • "ウミユリ海底譚" → automatically finds chart versions in database
  • If both [DX] and [STD] exist, aliases are added to both versions
Pro Tips:
  • Base Names Work: Use song names without [DX]/[STD] tags for easier uploading
  • Download Reference: Check the Chart Database download to see exact song titles if needed
  • Batch Processing: Upload multiple songs at once for efficiency
  • Auto-Detection: System automatically finds all chart variants
